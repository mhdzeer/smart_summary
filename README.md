# Smart Arabic Video Summarizer (V 4.3)

هذه الإضافة المزدوجة (ووردبريس + إضافة متصفح) صممت خصيصاً لتجاوز جميع عقبات حظر الـ API والحصة المجانية لـ Google Gemini. تعتمد الفكرة على استخدام "متصفحك الشخصي" كبوابة للذكاء الاصطناعي، مما يضمن عملاً مستقراً ومجانياً 100%.

## ✨ المميزات:
- **تجاوز حظر الـ IP**: يعمل التلخيص عبر متصفحك مباشرة وليس عبر السيرفر.
- **أتمتة كاملة**: ضغطة زر واحدة في ووردبريس تفتح Gemini في الخلفية، تلخص، ثم تغلق التبويب وتلصق النتيجة وتنسقها تلقائياً.
- **تنسيق ذكي**: تحويل ردود Gemini (Markdown) إلى مقال HTML مرتب (H3, Bold, Lists).
- **تنظيف النواتج**: حذف الروابط، عدد المشاهدات، وأسماء القنوات ليبقى المقال نظيفاً وجاهزاً للنشر.

## 🚀 طريقة التركيب:

### 1- إضافة ووردبريس (WordPress Plugin):
1. ارفع ملف `smart-video-summarizer.php` لمجلد `wp-content/plugins/` أو ارفعه كملف مضغوط من لوحة التحكم.
2. قم بتفعيل الإضافة.
3. بداخل أي مقال يحتوي على رابط يوتيوب، ستجد صندوق (Meta Box) أسفل المحرر لبدء التلخيص.

### 2- إضافة المتصفح (Chrome Extension):
1. افتح متصفح Google Chrome (أو أي متصفح مبني عليه).
2. انتقل إلى `chrome://extensions/`.
3. قم بتفعيل **Developer Mode** (وضع المطور) من الزاوية العلوية اليمنى.
4. اضغط على **Load unpacked** (تحميل إضافة غير محزمة).
5. اختر مجلد `extension` الموجود بداخل هذا المشروع.
6. تأكد من تثبيت أيقونة الإضافة بجانب شريط العنوان.

## 🛠️ كيف تستخدم النظام؟
1. افتح صفحة تحرير أي مقال في ووردبريس يحتوي على رابط فيديو.
2. اضغط على زر **"🚀 ابدأ التلخيص (أتمتة كاملة)"**.
3. انتظر بضع ثوانٍ (سيفتح تبويب Gemini ويغلق تلقائياً في الخلفية).
4. ستظهر لك رسالة نجاح ويتم إدراج المقال الملخص في أسفل محتواك فوراً.
